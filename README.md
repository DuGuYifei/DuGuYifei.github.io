# DuGuYifei 个人网站

基于 VitePress 的个人网站，模板来源：[vitepress-nav-template](https://github.com/maomao1996/vitepress-nav-template)

## 本地开发

### 环境要求

- Node.js 18+
- pnpm 10+

### 安装依赖

```bash
pnpm install
```

### 本地运行

启动开发服务器：

```bash
pnpm dev
```

访问 http://localhost:8732 查看网站

### 构建

构建生产版本：

```bash
pnpm build
```

### 预览构建结果

```bash
pnpm preview
```

访问 http://localhost:8730 查看构建后的网站

### 代码格式化

```bash
pnpm format
```
